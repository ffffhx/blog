> 先说明一下：下面提到的这些平台，都是笔者自己实际用过的，不是广告，也不是推广，大家完全可以按自己的情况自由选择其他平台。

## 第一部分：订阅 Claude Code

### 1. 账号注册

#### 1.1 推荐使用 Google 邮箱注册

- **步骤**：

1. 打开 Google 邮箱注册页面，填写相关信息创建新邮箱

#### 1.2 手机号验证解决方案

- **接码平台推荐**：
- [https://hero-sms.com/cn](https://hero-sms.com/cn)
- **操作步骤**：

1. 选择一个接码平台并注册账号
2. 充值平台余额（有点坑的是一次最少充 3 刀，一次接受验证码，美国号码的费用大概是 0.5 刀）
3. 选择美国手机号码（我有同事用波兰的、英国的也成功了，我为了稳健，选择了美国的）
4. 复制手机号码用于 Claude Code 注册
5. 接收验证码并完成验证

### 2. 订阅 Claude Code

这里有两种方案：

- 方案 A：直接购买国内外平台售卖的 Claude Code 官方礼品卡（不推荐，风险太高）
  这类礼品卡来源不明，可能是黑卡。一旦被官方发现，账号有被封的风险。方便是真方便，但为了账号安全，不建议这么做。
- 方案 B：Apple 美区账号订阅（推荐）

#### 2.1 购买美区 Apple ID

- **购买渠道**：
- [https://nf.video/](https://nf.video/)
- **注意事项**：
- 购买后及时修改密码

#### 2.2 购买 Apple 礼品卡

- **购买方式**：

1. 打开支付宝，将左上角的地点调到旧金山
2. 点击礼品卡，如图

{% asset_img figure-01.png %}

1. 找到苹果礼品卡，样式如图，或者直接在搜索框搜索 `apple`

{% asset_img figure-02.png %}

1. 选择合适的面额，Pro 一般是 20 刀，MAX 5x 为 100 刀，MAX 20x 为 200 刀，这里付款可以直接使用支付宝支付

{% asset_img figure-03.png %}

4. 完成支付
5. 获取礼品卡代码

#### 2.3 下载并订阅 Claude Code

- **操作步骤**：

1. 打开 App Store，并登录你购买的美区 Apple ID

{% asset_img figure-04.png %}

1. 点击“兑换充值卡或代码”

{% asset_img figure-05.png %}

3. 搜索 `Claude`，找到 `Claude by Anthropic`

{% asset_img figure-06.png %}

5. 下载、打开应用并登录账号
6. 进入订阅页面，选择订阅方案
1. 使用礼品卡代码进行支付

### 3. 代理搭建（其实非常简单，直接交给 AI 就可以了）

**为什么要自己搭建，我直接买一个梯子不行吗：**

因为 Claude 封控极其严格，如果你的 IP 不纯净，很多人共享，或者 IP 不稳定，一会儿美国一会儿日本，是非常容易被封号，所以我这儿选择买个美国住址 IP 的 VPS，然后自己搭建。

#### 3.1 购买美国 VPS 服务器

- **推荐服务商**：
- [https://lisahost.com/](https://lisahost.com/)
- **配置建议**：
- 本人购买的 88 元版本，晚上会有些慢，其他正常

#### 3.2 搭建代理服务

- **操作步骤**：

1. 打开 Claude Code（也可以先用中转站 Claude Code，例如 [https://api.mcxhm.cn](https://api.mcxhm.cn)），或者其他能操作本地环境的 AI 工具，比如龙虾也可以。
2. 输入 prompt：`使用 https://github.com/XTLS/Xray-core，帮我在这台美国 VPS 上搭建一个代理服务器`
3. Claude Code 会给你一个链接，然后会给你推荐几个客户端，你可以直接让它帮你下载并安装，本人用的是 V2RayU
4. 打开 V2RayU，复制链接，导入服务器配置，配置客户端连接信息

- **注意事项**：

1. 代理的客户端本人使用的是 V2RayU（尝试过 V2RayXS，但是不知道为什么不行）
2. 可能需要给终端也配置代理，具体怎么配置可以直接咨询 AI
1. 如果你的电脑是 Mac，要注意打开的 V2RayU 会在你的屏幕右上角，如果你右上角的应用太多可能会看不到 V2RayU。我当时遇到这样的问题，明明进程存在，但就是找不到，结果发现是太多被遮挡住了。

## 第二部分：订阅 Codex

Codex 这一部分其实可以说得很简单：不用搭建代理，其他跟 Claude Code 的购买方式完全相同就可以了。

也就是说，你直接复用上面第一部分里准备好的美区 Apple ID、Apple 礼品卡和应用内订阅流程即可，不需要再额外折腾一套代理环境。
